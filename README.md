**Elevating Customer Retention through Gamified Loyalty & Predictive Restocking**

## 🎀 The Problem
In the highly competitive beauty e-commerce space, **first-time buyer retention** is a critical challenge. Users often treat beauty apps as one-off purchase destinations rather than a daily habit. 

**Nykaa Glow Rewards** is designed to bridge this gap by transforming the post-purchase experience into a rewarding journey of self-care.

## 🚀 Key Features

### 1. 🏆 Gamified Loyalty Hub
Move beyond basic points. Users progress through tiers (Silver to Gold) by completing **Daily Glow Missions** like "Log in," "Write a Review," or "Try a Skincare Quiz."
*   **Visual Progress:** Real-time progress bar to the next tier.
*   **Instant Gratification:** Smooth animations and celebration modals upon earning points.

### 2. 🔄 Smart Reorder Center
Beauty products run out. Instead of letting users forget, our **Predictive Restock** feature tracks usage patterns and alerts users when they are "Running Low."
*   **One-Click Checkout:** Frictionless reordering of staples.
*   **Subscription Toggle:** Encourages recurring revenue with automated monthly deliveries.

### 3. 🧪 Personalized Recommendations
AI-driven product discovery tailored to the user's specific skin type (e.g., Oily Skin).
*   **Curated Feed:** High-fidelity UI with discount urgency and quick "Add to Bag" integration.

### 4. 📱 Mobile-First Native Experience
Built as a high-fidelity prototype that mimics a premium native app feel with:
*   Glassmorphism UI elements.
*   Custom smooth-scroll navigation.
*   Optimized mobile-safe layouts.

## 🛠 Tech Stack
*   **Frontend:** React 18, Vite
*   **Styling:** Tailwind CSS (Custom theme for Nykaa Brand Guidelines)
*   **Icons:** Lucide React
*   **Persistence:** LocalStorage (Maintains points and mission status across sessions)

## 🏁 Getting Started

### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Adityasri05/Nykaa-BeautyLoop.git
    cd Nykaa-BeautyLoop
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **View the app:**
    Open `http://localhost:5173` in your browser. Use **Inspect Element > Toggle Device Toolbar** for the best mobile-first experience.
